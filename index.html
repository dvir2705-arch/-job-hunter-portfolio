// Why I Built This

The Problem

  • Sending the same generic CV to every job listing
  • Manually tailoring each CV takes 30+ minutes
  • Losing track of applications across companies
  • Missing follow-up deadlines

The Solution

  • Claude AI adapts your CV per job in seconds
  • Automated scanning across 5 job sources
  • Full application lifecycle tracking
  • One CLI command: scan, filter, adapt, apply

// Core Features

🤖

AI CV Adaptation

Claude AI analyzes each job description and rewrites your CV to highlight the most relevant skills and experience. Output is a print-ready PDF.

🔍

Multi-Source Job Scanning

Scans LinkedIn, Workday, Amazon Jobs, and company career pages in parallel. AI-powered relevance filtering removes irrelevant matches.

✉

Cover Letter Generation

Generates tailored cover letters that adapt focus area by job type — software roles emphasize coding, RF roles highlight signals experience.

📊

Application Tracking

Tracks every application with status flow (applied → screening → interview → offer), follow-up reminders, and statistics dashboard.

👥

Multi-Profile System

Supports multiple user profiles with isolated data — designed for multi-user readiness with per-profile CV, applications, and settings.

🎯

Smart Relevance Filter

Two-tier filtering: fast keyword matching, then Claude AI deep-checks uncertain jobs. Adapts keywords from your profile's skills and target roles.

// Engineering Highlights

Parallel Job Scanning

ThreadPoolExecutor scans 5 sources concurrently with per-source caching (90-min TTL) and progressive CLI output.

Two-Tier AI Filtering

Fast keyword pre-filter eliminates obvious mismatches, then Claude AI deep-checks borderline jobs — balancing speed with accuracy.

Profile-Driven Design

All features derive behavior from the user profile — search queries, filter keywords, CV rules, and cover letter style adapt automatically.

Multi-Profile Isolation

Each profile gets its own data directory. Legacy single-user setups auto-migrate on first run. Built for multi-user readiness.

Robust Error Handling

JSON parse guards, graceful API failure fallbacks, cross-platform Chrome detection (Windows/macOS/Linux), malformed data recovery.

Test Coverage

334 tests across three layers: unit tests for models, integration tests for modules, and CLI-level tests using Click's CliRunner.
